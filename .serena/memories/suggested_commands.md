Key commands: `uv sync` to install deps, `uv run sophos-cli --help` or `uv run python -m sophos_cli --help` to run the CLI, `uv run pytest` for tests, `uv run ruff check .` for lint, `just analyze` for lint/format-check/type-check, `just format` to auto-format/fix, `just test` and `just test-cov` for testing. In this workspace, `.venv/bin/pytest`, `.venv/bin/ruff check .`, and `.venv/bin/pyright` also work directly.
