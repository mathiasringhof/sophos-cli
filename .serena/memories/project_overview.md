Python 3.12+ Typer-based CLI for Sophos Firewall automation. Uses src/ layout with domain commands under src/sophos_cli/commands, reusable logic in services, Pydantic models in models, and input parsing in io. Current public CLI has root commands version, test-connection, get-tag, plus a dns command group for DNSHostEntry CRUD/bulk workflows. Rich is used for console rendering, sophosfirewall-python is the SDK, and tests are pytest-based with mocked firewall clients.
