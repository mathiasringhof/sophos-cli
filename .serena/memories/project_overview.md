Python 3.12+ Typer-based CLI for Sophos Firewall automation. It uses a src layout under `src/sophos_cli` with lightweight root CLI wiring, domain command groups in `commands/`, reusable logic in `services/`, models in `models/`, input parsing in `io/`, and pytest coverage in `tests/`. The current command surface combines explicit LLM-friendly resource trees such as `dns` and `network` with SDK-backed escape hatches via visible `api` commands and a hidden `raw` group.
